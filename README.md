## Сборка <a name = "about"></a>


Создание виртуального окружения
```
$ python -m venv venv
```
```
$ venv\Scripts\activate
```
Установка требуемых зависимостей
```
$ pip install -r requirements.txt
```
Создание конфигурационного файла
```
$ touch config.py
```
Запуск
```
$ python main.py
```
